Inline Elements

<a> is an anchor (external hyperlink)

<a> is an anchor (internal hyperlink)

<abbr> is for abbreviations like FORTRAN (see also <acronym>)

<acronym> is for acronyms like FCS (see also <abbr>)

<b> is for bold text (deprecated, replaced by <strong>, see also <i>)

<big> is for big text (see also <small>)

<br> is a line break - this is before the break
And this is after the break

<cite> is for citations, like "Mad Magazine"

<code> InLineCodeSnippet()

<del> is for deleted (struck through) text (see also <ins>)

<em> is for emphasized text (replaces <i>, see also <strong>)

<i> is for italic text (deprecated, replaced by <em>, see also <b>)

<img> is to embed an image:

<ins> is for inserted text (see also <del>)

<kbd> is for "keyboard" text (see also <tt>)

<dfn> is for a definition

<q> is an inline quotation (unlike <blockquote>, which is a block element)

<samp> is for sample text

<small> is for small text (see also <big>)

<strong> is for strong text (replaces <b>, see also <em>)

<sub> is for subscripted text (see also <sup>)

<sup> is for superscripted text (see also <sub>)

<tt> is for "typewritten" text (see also <kbd>)

<var> is for variable or substituted text

<a> is an anchor (target of internal hyperlink)

Block Elements
